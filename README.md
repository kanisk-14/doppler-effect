# Doppler Effect — Interactive Explorer

## Overview

This project is a web-based interactive application designed to demonstrate and explain the Doppler Effect in wave physics. 
It provides conceptual explanations, mathematical formulations, real-time simulations, and practical applications to enhance understanding.

---
## Features
* Conceptual explanation of the Doppler Effect
* Presentation of the master equation and related formulae
* Interactive simulation of wave propagation
* Visualization of Mach cone and supersonic motion
* Real-world applications of the Doppler Effect
* Quiz section for self-assessment

## Description

The application demonstrates how the observed frequency of a wave changes due to the relative motion between 
the source and the observer. 
The simulator allows users to adjust parameters such as source velocity, observer velocity, frequency, 
and speed of sound to observe changes in real time.
---
## Limitations

* The simulation uses simplified physics models for visualization purposes
* Environmental factors are not considered
* The speed of sound is approximated
